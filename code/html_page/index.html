Metodologia

Métodos ágeis

Utilizar métodos ágeis significa colocar as pessoas em primeiro lugar. O sucesso de um projeto de software começa com um time qualificado e que ama o que faz. Cliente como parte da equipe, comunicação aberta e foco no resultado complementam a receita.

Acreditamos que não existem dogmas em desenvolvimento de software e que o mais importante é o aprendizado contínuo.


Qualidade

A experiência de uso determina a qualidade de um software. Por isto, dedicamos a maior parte de nossos esforços para a criação de interfaces bem projetadas e desenhadas.

Além disso, utilizamos uma série de técnicas para criar sistemas de alta qualidade. Controle de versões com git, testes automatizados, verificação da cobertura de testes, integração contínua e deployment automatizado são algumas delas.


Tecnologia

Empresas de tecnologia não devem ter medo de experimentar. Estamos sempre estudando novas ferramentas e testando novas plataformas. Hoje, trabalhamos principalmente com Ruby on Rails e PostgreSQL.

Nosso cinto de utilidades conta também com linguagens como C, JavaScript (Node.js), Python, Erlang e Haskell.
